# Simple CV with React and Sass
## Usage

### npm v12.6+

```
npm install
npm start
```

#### yarn v1.22+

```
yarn install
yarn start
```

## How to edit

- Create or edit files in `src/data` with your information
- If you create your own folder on `src/data` you need to edit the import in `src/pages/MainPage.tsx`, by default it is `src/data/js`
- If you want you can change the color palette in `src/utils/sass/vars.scss`

### And that's all! You get a simple CV now.

## To close dark mode button

- hover on bottom of dark mode button and click on close button
